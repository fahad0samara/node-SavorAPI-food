Usage Instructions

Welcome to our API! Below are the instructions on how to use it:

Base URL: https://savorapi.azurewebsites.net

1. Get all categories

Endpoint to retrieve all categories.

GET /categories

2. Get all products (without pagination)

Endpoint to retrieve all products without pagination.

GET /products

3. Get paginated products

Endpoint to retrieve products with pagination support.

GET /products?page=1&pageSize=10

4. Get a single product by ID

Endpoint to retrieve a single product by its ID.

GET /products/:id

5. Create a new product

Endpoint to create a new product.

POST /products

Include the following JSON data in the request body:

{
    "name": "Product Name",
    "description": "Product Description",
    "price": 10.99,
    "originalPrice": 12.99,
    "discountPercentage": 15,
    "stockQuantity": 100,
    "isNewProduct": true,
    "category": "Category ID",
    "ingredients": ["Ingredient 1", "Ingredient 2"],
    "allergens": ["Allergen 1", "Allergen 2"],
    "servingSize": "100g",
    "calories": 250,
    "cookingInstructions": "Cooking instructions...",
    "dietaryRestrictions": ["Vegetarian", "Gluten-free"],
    "expirationDate": "2024-12-31"
}

6. Update a product by ID

Endpoint to update an existing product by its ID.

PUT /products/:id

Include the following JSON data in the request body to update the product:

{
    "name": "Updated Product Name",
    "description": "Updated Product Description",
    "price": 12.99,
    "originalPrice": 14.99,
    "discountPercentage": 20,
    "stockQuantity": 150,
    "isNewProduct": false,
    "category": "Updated Category ID",
    "ingredients": ["Updated Ingredient 1", "Updated Ingredient 2"],
    "allergens": ["Updated Allergen 1", "Updated Allergen 2"],
    "servingSize": "150g",
    "calories": 300,
    "cookingInstructions": "Updated Cooking instructions...",
    "dietaryRestrictions": ["Vegetarian", "Gluten-free", "Dairy-free"],
    "expirationDate": "2025-12-31"
}

7. Delete a product by ID

Endpoint to delete an existing product by its ID.

DELETE /products/:id
