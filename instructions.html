Usage Instructions

Welcome to our API! Below are the instructions on how to use it:

Base URL: https://savorapi.azurewebsites.net

1. Get all categories

Endpoint to retrieve all categories.

GET /categories

2. Get all products (without pagination)

Endpoint to retrieve all products without pagination.

GET/products/all

3. Get paginated products

Endpoint to retrieve products with pagination support.

GET /products?page=1&pageSize=10

4. Get a single product by ID

Endpoint to retrieve a single product by its ID.

GET /products/:id

Fetch Data Using async/await

To fetch data from the API endpoints using async and await, you can use the following JavaScript code:

async function fetchData() {
            try {
                const response = await fetch('https://savorapi.azurewebsites.net/categories');
                const categories = await response.json();
                console.log('Categories:', categories);
        
                // Fetch products
                const productResponse = await fetch('https://savorapi.azurewebsites.net/products/all');
                const products = await productResponse.json();
                console.log('Products:', products);
            } catch (error) {
                console.error('Error fetching data:', error);
            }
        }
        
        // Call the fetchData function
        fetchData();

This code asynchronously fetches categories and products from the API endpoints using the fetch function and async/await syntax. It logs the retrieved data to the console.

Example Usage: Fetching data from an API to display a list of categories and products on a website.

Error Handling: If an error occurs during data fetching, the catch block logs the error to the console. Users can review the error message and troubleshoot accordingly.

For more information on async/await and the fetch API, refer to the MDN Web Docs.
